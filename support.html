Support

If you have questions about how to use PySB, we'd be happy to help. We recommend using our open chat channel on Gitter. It's accessible from your web browser, but you'll need a Github login.

Alternatively, if your question is more general (e.g. about numpy or scipy), we recommend posting a question to Stack Overflow and tagging it with "pysb".
